# Grapheme
A lightweight, portable, localized translation &amp; transcription tool written in C++

# Requirements
- Visual Studio 2026 (v145 toolset)
- Python 3
- Vulkan (version 1.3.280.0 or higher)

# Setup
- After cloning the repo, simply run the 2 batch scripts within the `Scripts` folder.
- Once finished, you can open up the solution file via VS 2026. Then, you can compile & run the application
